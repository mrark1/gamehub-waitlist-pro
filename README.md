# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/mrark1/gamehub-waitlist-pro.git
```

---

## Backend

```bash
cd server

npm install

npm run dev
```

Runs on

```
http://localhost:5000
```

---

## Frontend

```bash
cd client

npm install

npm run dev
```

Runs on

```
http://localhost:5173
```
